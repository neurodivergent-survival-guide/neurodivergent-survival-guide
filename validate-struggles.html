  • Condition Suffixes: Remove "(ADHD)", "(Autism)", "(Bipolar)", "(BPD)", or "(OCD)" from link text. Example: Change "Emotional Dysregulation (ADHD)" to just "Emotional Dysregulation"
  • Invalid Slugs: Ensure all linked struggles match exactly the slugs defined in _data/struggle_order.yml
  • Link Counts: Each struggle page must have exactly 4 or 5 related struggles (not "4-5", pick one number)
  • Missing Links: If a page shows 0 links, check that it has a <div class="related-struggles"> section with struggle-tag links
